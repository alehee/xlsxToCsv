# xlsxToCsv

Simple console *xlsx* to *csv* converter written in .NET 6

Put your *xlsx* files to the 'in' folder and run the program. After execution your *csv* files will be outputed to the 'out' directory.

## Used technology
* .NET 6
* NuGet: ClosedXML
